An IDE is required to run this game, as well as the Java Development Kit. 
Download both here:
    JDK (java Development Kit): https://www.oracle.com/java/technologies/downloads/
    IDE (Integrated Development Environment): Visual Studio Code: https://code.visualstudio.com/download
    

To run Sigian Conflict , download this code through a ZIP file. 
Then open Visual Studio Code, your IDE.
At the top left, click File, then Open Folder.

Find Sigian-Conflict on your device, then click "Open".
Then, navigate to:
src/game/Driver.java
and then run the java file.
